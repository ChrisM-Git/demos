The Prisma AIRS Python SDK (pan-aisecurity) lets you embed runtime security scanning directly into your own Python applications — chatbots, content moderation systems, AI agents, or any LLM-powered app.

Sync & Async

Supports synchronous and asynchronous scanning with concurrent capabilities and flexible retry strategies (default: 5 retries).

Full Coverage

Handles authentication, error management, and data parsing. Works across platforms with comprehensive type definitions.

Installation

Terminal
# Create and activate a virtual environment
python3 -m venv --prompt aisec-api-sdk-${USER} .venv && source .venv/bin/activate

# Install the SDK
python3 -m pip install pan-aisecurity

Configuration

Python
# Option 1: Environment variable
export PANW_AI_SEC_API_KEY=YOUR_API_KEY_GOES_HERE

# Option 2: Pass directly in code
aisecurity.init(api_key="YOUR_API_KEY_GOES_HERE")

# Optional: custom endpoint (default is US)
aisecurity.init(api_endpoint="https://service-de.api.aisecurity.paloaltonetworks.com")

# Set your security profile (name OR id — not both required)
ai_profile = AiProfile(profile_name="YOUR_PROFILE_NAME")
# or
ai_profile = AiProfile(profile_id="YOUR_PROFILE_ID")
